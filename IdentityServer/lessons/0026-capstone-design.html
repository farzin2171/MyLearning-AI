What We're Building — "WorkSpace"

WorkSpace is a fictional productivity platform: a web app for users, two protected backend APIs, a reporting service that calls the APIs without a user, and one Duende IS instance that secures all of it.

Every Design Decision — One Table

WhatDecisionWhy
Clients
WorkSpace Web GrantTypes.Code + PKCE, RequireClientSecret = false Public client running in user's browser flow; no secret needed
Reports Worker GrantTypes.ClientCredentials + hashed secret Machine-to-machine; no user present
Scopes
Calendar API surface calendar.read, calendar.write Separate read/write so clients can be scoped minimally
Contacts API surface contacts.read, contacts.write Same pattern — each API owns its scope namespace
Identity scopes openid, profile, email Standard OIDC; web app needs user identity in ID token
Claims
Access token carries sub, scope, tenant_id, role APIs need tenant to enforce row-level isolation
ID token carries sub, name, email Web app needs to display user info; auth claims stay in access token
Authorization policies
Calendar API — read RequireClaim("scope", "calendar.read") Simple scope policy — any tenant can read their own events
Calendar API — delete Custom EventOwnerRequirement (imperative) Must check the specific event's tenant_id against token's tenant_id
Reports Worker Scope only — no user context in token Machine tokens carry scopes, not user roles
Users
User store (development) TestUserStore (QuickStart UI) Hard-coded users; swap for real DB in production
Profile enrichment IProfileService reads tenant_id + role from test dict Both Calendar and Contacts APIs need these to enforce isolation
Token signing
Development AddDeveloperSigningCredential() Throwaway key — fine for local dev and tests
Production (future) AddSigningCredential(X509Certificate2) Stable key surviving restarts; JWKS cache remains valid

The Full Config.cs Sketch

Config.cs — complete capstone configuration
public static class Config
{
    public static IEnumerable<IdentityResource> IdentityResources => new[]
    {
        new IdentityResources.OpenId(),
        new IdentityResources.Profile(),
        new IdentityResources.Email(),
    };

    public static IEnumerable<ApiScope> ApiScopes => new[]
    {
        new ApiScope("calendar.read",   "Read calendar events",
                     userClaims: new[] { "tenant_id", "role" }),
        new ApiScope("calendar.write",  "Write calendar events",
                     userClaims: new[] { "tenant_id", "role" }),
        new ApiScope("contacts.read",   "Read contacts",
                     userClaims: new[] { "tenant_id" }),
        new ApiScope("contacts.write",  "Write contacts",
                     userClaims: new[] { "tenant_id" }),
    };

    public static IEnumerable<Client> Clients => new[]
    {
        // Interactive web app
        new Client
        {
            ClientId            = "workspace-web",
            AllowedGrantTypes   = GrantTypes.Code,
            RequirePkce         = true,
            RequireClientSecret = false,
            RedirectUris        = { "https://localhost:5002/signin-oidc" },
            PostLogoutRedirectUris = { "https://localhost:5002/signout-callback-oidc" },
            AllowOfflineAccess  = true,
            AllowedScopes       =
            {
                "openid", "profile", "email",
                "calendar.read", "calendar.write",
            }
        },

        // Background reporting service
        new Client
        {
            ClientId          = "reports-worker",
            AllowedGrantTypes = GrantTypes.ClientCredentials,
            ClientSecrets     = { new Secret("worker-secret".Sha256()) },
            AllowedScopes     = { "calendar.read", "contacts.read" }
        }
    };
}
