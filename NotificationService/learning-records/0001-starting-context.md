# Starting Context: Taking Ownership of the Notification Service

The user is an Equisoft developer taking ownership of the Notification Service. They have no stated prior knowledge of this specific service and want to understand it end-to-end — starting with the overall flow before diving into code details.

**Implications:** Begin with the high-level architecture and the two core flows (connection registration + notification delivery), then layer in SignalR internals, Redis patterns, and resilience. Do not assume deep SignalR knowledge — establish the basics in Lesson 1 before going deeper.
