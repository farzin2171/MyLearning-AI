Connection Registration (shared by both flows)

// 1. Browser opens WebSocket
GET /hubs/notifications?access_token=...

// 2. Hub assigns ConnectionId, joins group
OnConnectedAsync() → Groups.AddToGroupAsync(ConnectionId, "Authenticated SignalR Users")

// 3. Browser registers applicationId
SetApplicationId(appId) → KeepAliveManagementService.LinkAsync(appId, ConnectionId, UserId)

// 4. Stored in Redis:
Key:   appId.ToString()
Value: [{ connectionId, userId, applicationId }]
TTL:   3600s

The Single-Connection Guarantee

When LinkAsync is called for a userId that already has an entry under the same applicationId, the old entry is replaced (not accumulated). One active connection per userId + applicationId at all times. A browser refresh replaces the old record automatically.
