# Starting context established

The user is a developer at Equisoft who works with the Authorization Service as part of their core responsibilities. They described it as "one of the most important microservices we have," indicating professional stakes — this is not academic learning but operational knowledge. Familiarity with C# and .NET is assumed.

**Implications:** Lessons should connect directly to the real codebase. Explanations can skip language basics and go straight to architectural and domain concepts. The user needs the depth of a lead engineer, not an introduction.
